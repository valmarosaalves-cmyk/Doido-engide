## A rewrite of FNF: Doido Engine

This branch contains an early, in development version of FNF: Doido Engine, completely rewritten from scratch. It is VERY unstable and does not have most features you would expect yet, so it is not ready for use with mods, and we would appreciate any help you could give us, be it playtesting or contributing towards the code.

## Features

- Massively Improved performance
- ZIndex layering
- Redone Android support
- Pooling for notes, splashes, ratings etc for better performance
- New Chart Format
- Linear BPM Changes (UNTESTED)
- New Assets and Caching systems (with support for GPU caching)
- Full Flixel 6.x support
- Many rewritten classes

## Why did you restart development?

Mostly for fun. Also because Doido Engine v3.x.x was starting to feel a little bloated and we wanted to be able to structure the engine better and try to have better performance (which was one of the initial goals of Doido Engine and we gradually forgot about!).

## Contributing

Over the coming days we are looking into writting a full contributing guide including what we need help with, but if you find anything major missing, please create a pull request!

### Libraries

To install the correct libraries, you can run your platform's respective script in `art/setup/`. You can also install manually by checking the `.hmm` file.
